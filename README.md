# FlavourVerse
FlavourVerse is a full-stack web application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js) that enables users to share both blogs and recipes.

## Features
* User Authentication: Provides secure login and signup functionality using Passport.js, featuring seamless direct authentication with Google.
* Social Networking: Users can follow others, comment on posts, and interact within the community.
* Profile Section: Users can view their profile, including their posts, followers, and following.
* Responsive Front-end: The front-end is fully responsive, ensuring compatibility across various devices.
* Efficient Server-side Operations: Robust server-side operations ensure smooth and reliable application performance.
* Robust Data Management: Efficient data handling and management using MongoDB.
* Powerful Back-end: Strong back-end functionalities developed with Node.js and Express.js.
